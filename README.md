# hiReact
Learning the back of React 

## 实现points

- 虚拟dom; 继承Element元素的处理
- dom中type, props, children处理
- 渲染render时虚实dom挂载到父节点 -> mountTo
- 状态管理state && setState
- 虚拟dom的diff算法，并进行dom更新 -> update


## 工程实现
- 解析自定义jsx: @babel/plugin-transform-react-jsx
- webpack-dev-server
- 测试jest

## 启动
- npm install
- npm run dev
- npm run test
